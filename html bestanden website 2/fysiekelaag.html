De fysieke laag

De fysieke laag van een computer bevat alle fysieke hardware die allemaal samenwerken om ervoor te zorgen dat de computer werkt.
Het zijn de onderdelen die de software aansturen en die de fysieke uitvoeringen van berekeningen en taken laten slagen.
Hieronder ga ik dieper in op de verschillende onderdelen van de fysieke laag.

Processor (CPU)

De Central Processing Unit is als het ware het hart van de computer. De CPU wordt vaak het brein van de computer genoemd, omdat
het alle instructies uitvoert die door de programma's worden gegeven.
Iedere keer dat jij iets op de computer doet (bijvoorbeeld een programma openen), voert de CPU miljoenen kleine
berekeningen uit om die taak te laten slagen.

Waar bestaat de CPU dan uit? De CPU bestaat uit kleine schakelingen die razendsnel werken. De snelheid van de CPU wordt
gemeten in gigahertz (GHz). Hoe hoger de gemeten snelheid, hoe sneller de CPU de taken kan uitvoeren

Moederbord

Het moederbord is een heel groot elektronisch bord in de computer waarin de CPU, het geheugen en alle
andere onderdelen zijn gezet. Deze onderdelen worden verbonden met elkaar door dunne elektrische draden of sporen, daardoor
kunnen ze gegevens uitwisselen.

Het moederbord is als een soort van snelweg waar alle onderdelen mee communiceren. Zonder het moederbord zouden alle
onderdelen niet kunnen samenwerken met elkaar.
